# setMethod( string $method )

Set the HTTP method used to submit the form.

Set the HTTP method used to submit the form.

Possible values are: - <code>post</code>: Corresponds to the HTTP POST method ; form data are included in the body of the form and sent to the server. - <code>get</code>: Corresponds to the HTTP GET method; form data are appended to the FormInterface::action() attribute URI as a query string, and the resulting URI is sent to the server.
