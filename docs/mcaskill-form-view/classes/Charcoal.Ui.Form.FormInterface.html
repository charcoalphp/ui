setMethod()

setMethod(string  $method) : \Charcoal\Ui\Form\FormInterface

Set the HTTP method used to submit the form.

Possible values are:

  • \FormInterface::HTTP_METHOD_POST: Corresponds to the HTTP POST method ; form data are included in the body of the form and sent to the server.
  • \FormInterface::HTTP_METHOD_GET: Corresponds to the HTTP GET method; form data are appended to the \FormInterface::action() attribute URI as a query string, and the resulting URI is sent to the server.

Parameters

string $method

The HTTP method, usually one of GET or POST.

Returns
